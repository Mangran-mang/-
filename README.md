## 项目介绍
本项目是一个基于 Python 的自动化工具，用于学习 Python 网络爬虫与自动化技术。通过模拟用户操作，实现了登录学习通、拉取题目、匹配答案、提交结果的完整流程。

**⚠️ 重要声明**：本项目仅用于 Python 技术学习与研究，禁止用于任何违反学习通用户协议或学校规定的场景，使用者需自行承担相关责任。

## 技术栈
- Python 3.x
- Selenium（用于浏览器自动化操作）
- DeepSeek API（用于题目答案匹配）

## 核心流程
1. **登录**：通过 Selenium 模拟用户登录，获取登录后的 Cookie/Token
2. **拉取题目**：访问作业页面，解析题目内容（单选/多选/判断等）
3. **匹配答案**：将题目发送至 DeepSeek API，获取答案建议
4. **提交结果**：根据 API 返回的答案，自动选择并提交作业
